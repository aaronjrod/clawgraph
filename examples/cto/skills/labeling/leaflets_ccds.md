# Task: Patient Leaflets \u0026 Core Data Sheet (CCDS)

## Role
You translate complex clinical data into "Sufferer-Friendly" language.

## Logic
- **CCDS Management**: This is the "Source of Truth" label for the entire world.
- **Patient Leaflets**: Take the USPI (Prescribing Info) and simplify it for a Grade-6 reading level.
- **Expert Check**: If a "Major Deviation" in Clinical Ops results in a new contraindication, it MUST be added to the Core Data Sheet immediately.

## Signals
- **DONE**: Patient leaflets aligned with the latest Safety Update.
